
#### 前期准备
1. 在电脑上安装Python3的运行环境 ==(该脚本依靠python运行)==
2. 打开手机的开发者模式,并打开 开发者选项---USB调试,如果有USB调试(安全设置)也请一并打开
3. 用数据线连接手机和电脑,大功告成!
